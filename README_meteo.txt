SIGNIFICADO DE VARIABLES (HORARIAS Y DIARIAS)
TA = Temperatura de Aire (ºC)
WS_MAX = Ráfagas de viento (m/s)
HR = Humedad Relativa (%)
WD = Dirección del viento (º)
WS = Velocidad de viento (m/s)
RG = Radiación Global (W/m2)
RN = Radiación Neta (W/m2)
timestamp = Tiempo de medición (YYYY-mm-dd HH:MM:SS)
ingestion_time = Tiempo de creación de la data (YYYY-mm-dd HH:MM:SS)
source = Fuente de datos
station = Nombre de sitio
